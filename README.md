 Weather forecast Application 

The Weather Forecast Application is a simple and interactive web app that displays real-time weather information for any city worldwide. It retrieves weather data from a public weather API and presents it in a clean, responsive interface, making it easy for users to quickly check temperature, humidity, wind speed, and weather conditions.

# Features

1. Search weather for any city globally

2. Real-time temperature display

3. Shows humidity and wind speed

4. Error handling for invalid or empty inputs

5. Responsive and user-friendly UI

# Technologies Used

1.HTML

2.CSS

3.JavaScript

4.Weather API (OpenWeather or similar)
